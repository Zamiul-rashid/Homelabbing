## Step-by-Step Onboarding Walkthrough

### 1. Storage & Environment Setup
Verify that your host machine has `/srv/media` created (our automated `bootstrap.sh` script does this for you). Place at least one test video file inside `/srv/media/movies` so Jellyfin has something to index right away.

```bash
# Verify your UID and GID match your quickstart/.env file
id -u
id -g
```

### 2. Launching the Container
Navigate to the `01-media-server/` directory inside `quickstart/` and launch the stack:

```bash
cd quickstart/01-media-server
docker compose up -d
```

### 3. First-Time UI Setup Walkthrough
Once running, open your web browser to `http://YOUR_SERVER_IP:8096` and follow these 6 straightforward steps:

1. **Language Selection:** Choose **English** (or your native language) and click **Next**.
2. **Admin Account Creation:** Enter your preferred admin username and a strong password.
3. **Add Media Libraries:** Click **+ Add Media Library**:
   - **Content type:** Select `Movies`.
   - **Folders:** Click `+` and choose `/data/media/movies` (or `/data/media`).
   - Click **OK** and then **Next**.
4. **Metadata Settings:** Leave preferred language and country at defaults so Jellyfin automatically fetches movie posters, ratings, and plot summaries from TMDB.
5. **Remote Access:** Leave **Allow remote connections to this server** checked. Click **Next**, then **Finish**.
6. Sign in with your new credentials and enjoy your personal Netflix!
