## What You'll Have When Done
An automated media pipeline where browsing and requesting a movie or TV show on **Jellyseerr** automatically triggers search indexers in **Prowlarr**, sends the optimal quality torrent to **qBittorrent**, monitors progress in **Radarr/Sonarr**, and cleanly renames and moves the finished file straight into `/data/media` for Jellyfin to stream!

## The 5 Synchronized Containers

When you run this stack, you unlock five specialized tools working in harmony inside `stacks/arr-stack/`:

1. **qBittorrent (`:8080`)** — High-performance download engine with web interface and hardlink support.
2. **Prowlarr (`:9696`)** — Indexer manager that syncs search providers directly to Radarr and Sonarr.
3. **Radarr (`:7878`)** — Movie monitoring, searching, downloading, and automated renaming.
4. **Sonarr (`:8989`)** — TV show and anime monitoring with season/episode tracking.
5. **Jellyseerr (`:5055`)** — Request portal where family and friends can discover and request content.

## Step-by-Step Configuration Guide

### 1. Launching the Stack
Navigate to `stacks/arr-stack/` and start all 5 containers:
```bash
cd stacks/arr-stack
docker compose up -d
```

### 2. Connecting Prowlarr (`http://YOUR_IP:9696`)
1. Click **Add Indexer**, search for public indexers (e.g., `1337x`, `EZTV`), and click **Save**.
2. Go to **Settings → Apps → + Add**:
   - Add **Radarr** (`http://radarr:7878`) and paste your Radarr API key (found in Radarr under Settings → General).
   - Add **Sonarr** (`http://sonarr:8989`) and paste your Sonarr API key.
3. Click **Sync App Indexers** — your search providers are now automatically linked!

### 3. Linking Radarr & Sonarr to qBittorrent
In both Radarr (`http://YOUR_IP:7878`) and Sonarr (`http://YOUR_IP:8989`):
1. Go to **Settings → Download Clients → + Add → qBittorrent**:
   - **Host:** `qbittorrent`
   - **Port:** `8080`
2. Click **Test** and **Save**.

### 4. Connecting Jellyseerr (`http://YOUR_IP:5055`)
1. Open the Jellyseerr dashboard and log in using your Jellyfin account (`http://jellyfin:8096`).
2. Add your Radarr (`http://radarr:7878`) and Sonarr (`http://sonarr:8989`) endpoints using their API keys. You're ready to request!
